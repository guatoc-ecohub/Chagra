## Firma

La firma de este CLA se realiza por uno de los siguientes mecanismos:

1. **Por Pull Request (mecanismo principal)**: comentar en el PR exactamente la frase
   > `I have read the CLA Document and I hereby sign the CLA`
   El bot `contributor-assistant/github-action` registra la firma con su login de GitHub, timestamp ISO-8601 y el SHA del CLA vigente en `signatures/cla.json` (rama `cla-signatures`).
2. **Por correo certificado** (mecanismo alternativo, solo para empresas/contribuyentes corporativos): enviar el ICLA firmado a la dirección postal indicada en el README del repositorio. En este caso se acepta el Apache ICLA original v2.0 con el nombre del proyecto reemplazado por "Chagra / Guatoc Ecohub".

Una vez firmado, el CLA aplica a **todas sus contribuciones futuras** al Proyecto bajo el mismo login de GitHub o nombre legal, sin requerir refirma por PR.
